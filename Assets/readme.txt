///////////////////////////
///// Coin Pusher //////
///////////////////////////

To change the prices you can add any prefab that contains a collider and a rigidbody to
the pricespawner script in the main camera.

///////////////////////////

By changing the variables in the CoinManager you can start the game with more coins.

///////////////////////////

By changing the variables in the Spawnpoint you can change the time that it takes to get a new coin.
